# Mentify

A comprehensive mental health and wellness platform for students.

## Live Deployment

- **Frontend**: https://mentifyapp.vercel.app/
- **Backend**: https://mentify-backend.onrender.com

## Tech Stack

- Frontend: React + Vite
- Backend: Node.js/Express
- Database: MongoDB
- Deployment: Vercel (Frontend) + Render (Backend)
